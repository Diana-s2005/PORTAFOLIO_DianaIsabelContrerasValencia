#Corte - 03 - Proyecto 03 – Portafolio Web en WordPress

## Objetivo del Proyecto

Desarrollar un sitio web tipo portafolio utilizando WordPress que permita presentar información personal, habilidades, proyectos académicos y servicios relacionados con el desarrollo web, aplicando herramientas de gestión y personalización de contenido.

## Problema que Resuelve

El proyecto proporciona una plataforma digital para mostrar de manera organizada información profesional, proyectos realizados y servicios ofrecidos, facilitando la creación de una presencia en línea accesible y atractiva para posibles empleadores, clientes o colaboradores.

## Tecnologías Utilizadas

* WordPress
* PHP
* HTML5
* CSS3
* Astra Theme
* Contact Form 7
* XAMPP
* MySQL

## Conceptos Aplicados

* Gestión de contenido con WordPress
* Personalización de temas
* Configuración de menús de navegación
* Diseño web responsivo
* Administración de sitios web
* Uso de plugins
* Organización de contenido digital
* Formularios de contacto
* Diseño y experiencia de usuario
* Publicación de contenido web

## Descripción del Funcionamiento

El sitio web fue desarrollado utilizando WordPress como sistema gestor de contenidos y está compuesto por cinco páginas principales:

1. Inicio.
2. Sobre mí.
3. Portafolio.
4. Servicios.
5. Contacto.

La página de inicio presenta información general y habilidades relacionadas con el desarrollo web. En la sección "Sobre mí" se muestra información académica y objetivos profesionales. El apartado "Portafolio" contiene proyectos y prácticas desarrolladas durante la carrera. La sección "Servicios" describe actividades relacionadas con el desarrollo web y bases de datos. Finalmente, la página "Contacto" incorpora un formulario creado mediante el plugin Contact Form 7 para facilitar la comunicación con los visitantes.

El diseño visual fue implementado utilizando el tema Astra, el cual permite una amplia personalización y adaptación a diferentes dispositivos.

## Instrucciones de Ejecución

1. Instalar WordPress en un servidor local o servicio de hosting.
2. Configurar la base de datos correspondiente.
3. Instalar y activar el tema Astra.
4. Crear las páginas principales del sitio:

   * Inicio
   * Sobre mí
   * Portafolio
   * Servicios
   * Contacto
5. Configurar los menús de navegación.
6. Instalar y configurar el plugin Contact Form 7.
7. Personalizar colores, tipografías e imágenes del sitio.
8. Publicar el contenido y verificar el funcionamiento en diferentes dispositivos.

## Reflexión Personal

### ¿Qué aprendí?

Aprendí a utilizar WordPress como una herramienta profesional para la creación y administración de sitios web. También adquirí experiencia en la personalización de temas, instalación de plugins, configuración de menús y organización de contenido, además de reforzar conocimientos relacionados con diseño responsivo y experiencia de usuario.

La parte más compleja fue la personalización visual del sitio y la configuración adecuada de algunos elementos del tema y los plugins. También representó un reto seleccionar y organizar el contenido para que el portafolio mantuviera una apariencia profesional y ordenada.

Me gustaría ampliar el contenido del portafolio agregando más proyectos y evidencias de aprendizaje. Asimismo, mejoraría el diseño visual incorporando elementos más dinámicos, optimización para dispositivos móviles y nuevas funcionalidades que permitan una mejor interacción con los visitantes.
