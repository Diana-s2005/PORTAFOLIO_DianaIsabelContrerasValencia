## Proyectos Incluidos

### Proyecto 01 – Mini Página Web

Aplicación web desarrollada para la gestión de usuarios, contactos y citas. 
El proyecto permite registrar información mediante formularios web, aplicar validaciones básicas y almacenar datos utilizando PHP y bases de datos, fortaleciendo los conocimientos fundamentales del desarrollo de aplicaciones web.

### Proyecto 02 – Sistema de Gestion de Citas

Aplicación web desarrollada en PHP y MySQL bajo el patrón MVC que permite a los usuarios registrarse, iniciar sesión y administrar citas de manera eficiente. 
El sistema incluye funciones CRUD, autenticación segura mediante cifrado de contraseñas y una interfaz amigable para facilitar la gestión de información

### Proyecto 03 – Consumo de API REST con The Dog API

Proyecto desarrollado en PHP utilizando cURL para consumir servicios REST. 
La aplicación obtiene información de razas caninas desde The Dog API y muestra imágenes dinámicamente, permitiendo comprender el uso de solicitudes HTTP, manejo de JSON e integración de servicios externos en aplicaciones web.

### Proyecto 04 – Portafolio Web en WordPress

Sitio web tipo portafolio creado con WordPress y el tema Astra. Incluye secciones de presentación personal, información académica, proyectos realizados, servicios y formulario de contacto. 
El proyecto permitió aplicar conocimientos de administración, personalización y diseño responsivo de sitios web
