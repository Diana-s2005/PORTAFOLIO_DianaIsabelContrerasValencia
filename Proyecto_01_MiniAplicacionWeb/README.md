# Corte 01 - Proyecto 01 - Mini Aplicación Web

bjetivo del Proyecto

Desarrollar una aplicación web básica que permita registrar usuarios, contactos y citas, aplicando los fundamentos del desarrollo web y la interacción con bases de datos.

Problema que Resuelve

El proyecto permite organizar información personal mediante el registro de usuarios, contactos y citas, facilitando el almacenamiento y consulta de datos de forma estructurada.

Tecnologías Utilizadas
HTML5
PHP
JavaScript
phpMyAdmin
Conceptos Aplicados
Formularios HTML
Validación de datos con JavaScript
Programación en PHP
Conexión a bases de datos
Envío de datos mediante métodos POST
Organización de proyectos web
Descripción del Funcionamiento

La aplicación presenta una página principal que permite acceder a tres módulos:

Registro de usuarios.
Registro de contactos.
Registro de citas.

El módulo de usuarios incluye validaciones para verificar que los campos obligatorios sean completados antes de enviar la información. Los datos capturados son procesados mediante scripts PHP y almacenados en la base de datos.

Instrucciones de Ejecución
Instalar XAMPP o un servidor local compatible con PHP.
Colocar la carpeta del proyecto dentro del directorio del servidor web.
Configurar la base de datos y la conexión correspondiente.
Iniciar Apache y MySQL.
Abrir el navegador y ejecutar el archivo index.php.
Reflexión Personal
¿Qué aprendí?

Aprendí a desarrollar una aplicación web básica utilizando PHP, formularios HTML y conexión con bases de datos, además de organizar el proyecto mediante una estructura adecuada.

¿Qué fue difícil?

La configuración de la conexión a la base de datos y el procesamiento correcto de la información enviada desde los formularios.

¿Qué mejoraría?

Mejoraría el diseño visual de la aplicación, implementaría validaciones más completas y agregaría funciones para editar y eliminar registros.
